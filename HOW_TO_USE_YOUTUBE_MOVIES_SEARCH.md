# 🎬 How to Use YouTube Movies Search

## Overview
The YouTube Movies Search feature helps you discover and watch full-length movies available on YouTube. This includes both **free movies** and **YouTube Premium movies** (if you have a subscription).

## 🚀 Quick Start

### Basic Movie Search
```
YouTube movies [movie title]
```
**Examples:**
- `YouTube movies Avengers`
- `YouTube movies The Matrix`
- `YouTube movies Inception`
